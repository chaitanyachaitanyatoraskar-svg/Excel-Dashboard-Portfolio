# Sales Analytics Dashboard

## Project Overview
Built an interactive sales dashboard in Excel using Pivot Tables, Power Query and Slicers.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers

## Dataset
- 5,000 Sales Records
- 500 Customers
- 50 Products
- 7 Plants

## KPIs
- Revenue
- Profit
- Quantity Sold
- Top Products
- Top Plants
